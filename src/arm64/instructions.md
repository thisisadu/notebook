1. 寄存器

   ```
   1. X0-X7 用作函数传参
   2. X9-X15用作临时寄存器，可以任何时候使用
   3. X16-X29也可以用作临时寄存器，但使用前需要保存，函数返回前恢复内容
   4. X29(FP) FramePointer
   5. X30(LR) Link Register 保存函数返回地址
   6. X31(SP/ZR) Stack Pointer 寄存器，ZERO寄存器
   7. PC program counter 程序计数器
   8. PSTATE 寄存器
   ```
